About

This project shows that blockchain is useful for more than cryptocurrencies, that it can be used by everyday people in their everyday lives, increasing security and transparency in the process.
Newestate allows the user to make a listing or browse listings by specific criteria. Make open house reservations by specifying listing type, home type, and move-in urgency. And create listings by specifying address, availability date, open house time, and house and leasing types.
